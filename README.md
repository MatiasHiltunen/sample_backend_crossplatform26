# Getting started

## Optional:

1. Install uv: https://docs.astral.sh/uv/getting-started/installation/
2. Clone the repo
3. Run uv sync
4. Activate venv
5. Start dev server with "fastapi dev main.py"

## Traditional:

1. Clone the repo
2. Create venv: "python -m venv .venv"
3. Activate it
4. Install deps: pip install -r requirements.txt
4. Start dev server with "fastapi dev main.py"
